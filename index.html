XCode Copy Build Result

Feb 2022 - XCode 13

Copy the library file

  • Select Target
  • Select Build Phases
  • Click the Plus Button
  • New Copy Files Phase
  • Destination: Absolute path
  • Path: $PROJECT_DIR/dist/$CONFIGURATION
  • Click the Plus button on the bottom and select the .a file to copy
  • Do this for all targets

Copy the header files

  • Select Target
  • Select Build Phases
  • Click the Plus Button
  • New Copy Files Phase
  • Destination: Absolute path
  • Path: $PROJECT_DIR/dist/$CONFIGURATION/include/library-name
  • Click the Plus button on the bottom and select all the .h files to copy
  • Do this for all targets

XCode - Command Line Build

Feb 2022 - XCode 13

List available SDKs

  • xcodebuild -showsdks

List schemes and other stuff

  • xcodebuild -list -workspace TestLib2.xcworkspace
  • xcodebuild -list -project TestLib2.xcodeproj

Clean

  • xcodebuild clean -workspace TestLib2-workspace.xcworkspace -sdk macosx11.3 -scheme testLib2-macOS
  • xcodebuild clean -project TestLib2.xcodeproj

Build

  • xcodebuild build -workspace TestLib2-workspace.xcworkspace -sdk macosx11.3 -scheme testLib2-macOS -configuration Debug
  • xcodebuild build -workspace TestLib2-workspace.xcworkspace -sdk iphoneos15.0 -scheme testLib2-iOS -configuration Debug
  • xcodebuild build -project TestLib2.xcodeproj -sdk iphoneos15.0 -target testLib2-iOS -configuration Debug

OpenGL App on MacOS

Feb 2022

Create a OpenGL App on MacOS using GLFW

Build GLFW for MacOS

  • Clone GLFW from GLFW Github into the folder glfw
  • GLFW - Build
  • GLFW uses GLAD for extensions GLAD - Multi-Language GL/GLES/EGL/GLX/WGL Loader-Generator based on the official specs
  • Use CMake to produce a XCode project in thew folder glfw-xcode - use that to build GLFW and run the examples

Create a new MacOS Game App

  • XCode / File / New Project
  • Game
  • Metal
  • Objective-C
  • Build and Run should work fine

Run as a GLFW App

  • Copy the triangle-opengl.cpp example file from GLFW to your new MacOS Game App
  • Remove everything except:
    • AppDelegate
    • Assets
    • Main (storyboard)
    • Entitlements
    • triangle-opengl.cpp
  • Linker - Add glfw-xcode/src/Debug/libglfw3.a
  • Include Files - add glfw/deps glfw/include
  • Build and Run should work fine
